Sobre a API CRUD CRUD

Este projeto utiliza a API CrudCrud para armazenar personagens de forma simples e rápida.

O que é a CrudCrud?

CrudCrud é uma API REST gratuita que permite criar, listar, editar e excluir recursos facilmente. Ela é ideal para prototipagem rápida, testes de aplicações, projetos pessoais, tutoriais e aplicações simples em produção.

Como funciona?

Cada entidade criada recebe automaticamente um campo único chamado _id, que é utilizado para leitura, atualização ou exclusão individual.

Método Recurso Descrição
POST /<recurso> Cria uma nova entidade com os dados em JSON
GET /<recurso> Retorna todas as entidades do recurso
GET /<recurso>/<id> Retorna uma entidade específica
PUT /<recurso>/<id> Atualiza uma entidade com base no ID
DELETE /<recurso>/<id> Exclui uma entidade com base no ID

Sobre o Endpoint

Ao iniciar o projeto, um endpoint REST exclusivo é gerado automaticamente para você pela CrudCrud.

Esse endpoint é usado como base para todas as requisições e deve ser combinado com o nome do recurso desejado, por exemplo:

https://crudcrud.com/api/<seu-endpoint>/personagens

⚠️ Atenção: esse endpoint tem duração de 24 horas e expira automaticamente após esse período. Sempre que necessário, gere um novo endpoint no site da CrudCrud.
